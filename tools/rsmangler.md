# rsmangler

## 简介
基于输入单词生成变体密码字典

## 核心参数
- 输入文件
- 输出文件

## 命令示例
```bash
# 密码变体生成
rsmangler --file input.txt --output wordlist.txt

# 所有变换
rsmangler --file input.txt --output wordlist.txt --all
```

## 使用场景
基于目标信息生成定制化密码字典
