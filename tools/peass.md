# peass

## 简介
权限提升自动化枚举脚本套件（包含 linpeas 和 winpeas）

## 核心参数
- 根据平台使用对应脚本

## 命令示例
```bash
# Linux 上运行 linpeas
./linpeas.sh

# Windows 上运行 winpeas
winpeas.exe

# 全面扫描
./linpeas.sh -a

# 快速扫描
winpeas.exe fast
```

## 使用场景
跨平台的权限提升枚举工具集
