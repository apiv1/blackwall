# dirbuster

## 简介
dirb 的 GUI 版本，支持多线程目录爆破

## 核心参数
- GUI 界面操作
- 支持多线程
- 内置多个字典

## 命令示例
```bash
# 启动 DirBuster GUI
dirbuster

# 命令行模式（使用 dirb 替代）
dirb http://target.com /usr/share/wordlists/dirb/common.txt
```

## 使用场景
需要图形界面进行目录爆破时使用
