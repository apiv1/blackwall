# recordmydesktop

## 简介
桌面录屏工具

## 核心参数
- `--no-sound`: 不录制声音
- `-o`: 输出文件

## 命令示例
```bash
# 录制桌面
recordmydesktop

# 指定输出文件
recordmydesktop -o output.ogv

# 录制特定区域
recordmydesktop --width 1920 --height 1080
```

## 使用场景
录制渗透测试过程，用于报告或演示
