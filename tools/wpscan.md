# wpscan

## 简介
WordPress 专用漏洞扫描器，枚举插件、主题、用户

## 核心参数
- `--url`: 指定目标 URL
- `--enumerate`: 枚举类型（u=用户，p=插件，t=主题）
- `--api-token`: 使用 API 获取漏洞信息

## 命令示例
```bash
# 基本用法
wpscan [参数]
```

## 使用场景
当目标是 WordPress 站点时的专用工具，能发现过时插件的已知漏洞
