# faraday-stop-gophish-stop

## 简介
停止 Faraday 和 Gophish 服务的脚本

## 核心参数
- 无参数

## 命令示例
```bash
# 停止 Faraday
faraday-stop

# 停止 Gophish
gophish-stop

# 或手动停止服务
pkill -f faraday
pkill -f gophish
```

## 使用场景
停止运行中的 Faraday 或 Gophish 服务
