# msfpc

## 简介
MSF Payload Creator，快速生成各种 payload

## 核心参数
- 指定 payload 类型和参数

## 命令示例
```bash
# 生成 payload
msfpc windows

# 指定 IP 和端口
msfpc windows 192.168.1.100 4444

# 生成 Linux payload
msfpc linux 192.168.1.100 4444
```

## 使用场景
比 msfvenom 更简单的 payload 生成工具
