# enum4linux

## 简介
Linux 下的 Windows/Samba 枚举工具，获取用户、共享、组信息

## 核心参数
- `-a`: 全面枚举
- `-U`: 枚举用户
- `-S`: 枚举共享
- `-G`: 枚举组

## 命令示例
```bash
# 基本用法
enum4linux [参数]
```

## 使用场景
当你发现 SMB 服务时，enum4linux 能快速枚举所有可用信息
