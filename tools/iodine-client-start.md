# iodine-client-start

## 简介
Iodine DNS 隧道客户端启动脚本

## 核心参数
无特定参数或使用默认参数

## 命令示例
```bash
# 启动 DNS 隧道客户端
iodine -f 192.168.99.1 tunnel.example.com

# 指定密码
iodine -f -P password 192.168.99.1 tunnel.example.com
```

## 使用场景
快速建立 DNS 隧道，适合受限网络环境
