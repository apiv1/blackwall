# nmap

## 简介
端口扫描和服务识别的行业标准，用于快速绘制目标网络拓扑

## 核心参数
- `-sS`: SYN 扫描（半开扫描），隐蔽且快速
- `-sV`: 服务版本探测，识别运行的具体服务版本
- `-O`: 操作系统指纹识别
- `-p-`: 扫描全部 65535 个端口
- `-A`: 激进扫描（包含 OS 检测、版本检测、脚本扫描、traceroute）

## 命令示例
```bash
# 快速扫描常用端口
nmap 192.168.1.100

# SYN 扫描（隐蔽扫描）
nmap -sS 192.168.1.100

# 服务版本探测
nmap -sV -p 80,443,22 192.168.1.100

# 全端口扫描 + 服务识别
nmap -p- -sV 192.168.1.100

# 激进扫描（OS检测、版本、脚本、路由）
nmap -A 192.168.1.100

# 扫描整个 C 段
nmap 192.168.1.0/24

# 使用 NSE 脚本扫描 SMB 漏洞
nmap -p 445 --script smb-vuln-* 192.168.1.100
```

## 使用场景
任何渗透测试的第一步，当你需要了解目标开放了哪些服务时的首选工具
