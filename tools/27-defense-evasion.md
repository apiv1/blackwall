# 防御规避 (Defense Evasion)

## macchanger
- **能干什么**: 修改网卡 MAC 地址
- **核心参数**:
  - `-r`: 随机 MAC 地址
  - `-a`: 随机同厂商 MAC
  - `-m`: 指定 MAC 地址
- **实战直觉**: 隐藏真实身份，绕过 MAC 地址过滤

## cryptsetup
- **能干什么**: Linux 磁盘加密工具（LUKS），加密和管理加密分区
- **核心参数**:
  - `luksFormat`: 格式化加密分区
  - `luksOpen`: 打开加密分区
  - `luksClose`: 关闭加密分区
- **实战直觉**: 保护敏感数据，加密存储设备，防止数据泄露

## exe2hex
- **能干什么**: 将 EXE 文件转换为十六进制格式，绕过文件上传限制
- **核心参数**:
  - `-x`: 输入 EXE 文件
  - `-p`: 输出 PowerShell 脚本
  - `-b`: 输出 Batch 脚本
- **实战直觉**: 当无法直接上传 EXE 时，转换为文本格式上传后再还原

## macchanger
- **能干什么**: 修改网卡 MAC 地址
- **核心参数**:
  - `-r`: 随机 MAC
  - `-a`: 随机同厂商 MAC
  - `-m`: 指定 MAC
- **实战直觉**: 绕过 MAC 地址过滤，隐藏真实身份，避免被追踪
