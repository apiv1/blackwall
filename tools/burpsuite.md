# burpsuite

## 简介
最强大的 Web 应用安全测试平台，集成代理、扫描、爆破等功能

## 核心参数
无特定参数或使用默认参数

## 命令示例
```bash
# 启动 Burp Suite（GUI）
burpsuite

# 使用命令行模式
burpsuite --project-file=project.burp

# 无头模式运行扫描
burpsuite --project-file=project.burp --config-file=config.json
```

## 使用场景
Web 渗透的瑞士军刀，当你需要拦截和修改 HTTP 请求时的必备工具
