# laudanum

## 简介
各种语言的注入文件和 webshell 集合

## 核心参数
- 选择合适的文件类型

## 命令示例
```bash
# 查看可用文件
ls /usr/share/laudanum/

# ASP 文件
ls /usr/share/laudanum/asp/

# PHP 文件
ls /usr/share/laudanum/php/

# JSP 文件
ls /usr/share/laudanum/jsp/

# 复制文件
cp /usr/share/laudanum/php/shell.php .

# 编辑配置
nano shell.php
```

## 使用场景
提供各种语言的 webshell 和注入文件
