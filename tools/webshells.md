# webshells

## 简介
各种语言的 webshell 集合

## 核心参数
- 选择合适的 webshell 文件
- 上传到目标服务器

## 命令示例
```bash
# 查看可用的 webshells
ls /usr/share/webshells/

# PHP webshell
ls /usr/share/webshells/php/

# ASP webshell
ls /usr/share/webshells/asp/

# JSP webshell
ls /usr/share/webshells/jsp/

# 复制 webshell
cp /usr/share/webshells/php/php-reverse-shell.php shell.php

# 编辑配置（IP 和端口）
nano shell.php
```

## 使用场景
选择合适的 webshell 进行文件上传攻击
