# metasploit-framework

## 简介
最强大的渗透测试框架，集成漏洞利用、后渗透、payload 生成

## 核心参数
- `msfconsole`: 启动控制台
- `search`: 搜索模块
- `use`: 使用模块
- `set`: 设置参数

## 命令示例
```bash
# 基本用法
metasploit-framework [参数]
```

## 使用场景
当你找到已知漏洞时，metasploit 通常有现成的 exploit 模块
