# unix-privesc-check

## 简介
Unix/Linux 权限提升检查脚本

## 核心参数
- `standard`: 标准检查
- `detailed`: 详细检查

## 命令示例
```bash
# 标准检查
./unix-privesc-check standard

# 详细检查
./unix-privesc-check detailed

# 输出到文件
./unix-privesc-check standard > output.txt
```

## 使用场景
Unix/Linux 系统权限提升向量检查
