# recon-ng

## 简介
模块化的 Web 侦察框架，类似 Metasploit 的侦察版

## 核心参数
- `marketplace install`: 安装模块
- `modules load`: 加载模块
- `run`: 执行模块

## 命令示例
```bash
# 基本用法
recon-ng [参数]
```

## 使用场景
自动化 OSINT 收集，整合多个数据源，适合大规模信息收集
