# binwalk3

## 简介
binwalk 的 Python 3 版本

## 核心参数
- 与 binwalk 相同

## 命令示例
```bash
# 基本扫描
binwalk3 firmware.bin

# 自动提取
binwalk3 -e firmware.bin

# 递归扫描
binwalk3 -M firmware.bin

# 显示熵值
binwalk3 -E firmware.bin
```

## 使用场景
固件分析和文件提取（Python 3 版本）
