# theHarvester

## 简介
从搜索引擎和公开数据源收集邮箱、子域名、主机信息

## 核心参数
- `-d`: 指定目标域名
- `-b`: 指定数据源（google、bing、linkedin 等）
- `-l`: 限制搜索结果数量

## 命令示例
```bash
# 基本用法
theHarvester [参数]
```

## 使用场景
社工前的情报收集，当你需要找到目标公司的员工邮箱列表时的首选
