# john

## 简介
经典的密码破解工具（John the Ripper），支持多种哈希格式

## 核心参数
- `--wordlist=`: 指定字典文件
- `--format=`: 指定哈希格式
- `--show`: 显示已破解的密码
- `--rules`: 使用规则变换

## 命令示例
```bash
# 基本用法
john [参数]
```

## 使用场景
当你没有 GPU 或需要破解特殊格式时，john 是可靠的选择
