# powersploit

## 简介
PowerShell 后渗透框架

## 核心参数
无特定参数或使用默认参数

## 命令示例
```bash
# PowerSploit 模块位置
ls /usr/share/powersploit/

# 使用示例（在 PowerShell 中）
# Import-Module /usr/share/powersploit/Recon/PowerView.ps1
# Get-NetDomain
```

## 使用场景
Windows 后渗透的瑞士军刀，包含提权、凭据提取、持久化等模块
