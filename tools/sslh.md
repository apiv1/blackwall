# sslh

## 简介
多协议端口复用工具，在同一端口上运行多个服务

## 核心参数
- `-p`: 监听端口
- `--ssh/--ssl`: 指定后端服务

## 命令示例
```bash
# 多协议复用
sslh -p 0.0.0.0:443 --ssh 127.0.0.1:22 --ssl 127.0.0.1:8443

# 前台运行
sslh -f -p 0.0.0.0:443 --ssh 127.0.0.1:22 --ssl 127.0.0.1:8443
```

## 使用场景
在 443 端口同时运行 HTTPS 和 SSH，绕过端口限制
