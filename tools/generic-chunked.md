# generic_chunked

## 简介
生成分块传输 shellcode

## 核心参数
无特定参数或使用默认参数

## 命令示例
```bash
# Metasploit payload 相关
# 通常在 msfvenom 中使用
msfvenom -p generic/custom PAYLOADFILE=payload.bin -f exe
```

## 使用场景
绕过 IDS/IPS 的分块传输 shellcode
