# sqlmap

## 简介
自动化 SQL 注入检测和利用工具，支持多种数据库

## 核心参数
- `-u`: 指定目标 URL
- `--dbs`: 枚举数据库
- `--tables`: 枚举表
- `--dump`: 导出数据
- `--batch`: 非交互模式

## 命令示例
```bash
# 基本用法
sqlmap [参数]
```

## 使用场景
当你发现可能存在 SQL 注入点时，sqlmap 能自动化完成从检测到数据提取的全过程
