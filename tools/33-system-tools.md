# 系统工具与配置 (System Tools)

## Kali 系统工具

### Kali Undercover Mode
- **能干什么**: 将 Kali 界面伪装成 Windows 10 外观
- **核心参数**: 直接运行切换模式
- **实战直觉**: 在公共场所使用 Kali 时，避免引起注意

### Kali Halloween Mode
- **能干什么**: Kali 的万圣节主题模式
- **核心参数**: 直接运行切换主题
- **实战直觉**: 娱乐功能，改变系统外观

### Kali HiDPI Mode
- **能干什么**: 高分辨率显示器优化模式
- **核心参数**: 直接运行调整 DPI 设置
- **实战直觉**: 在 4K 或高分辨率显示器上使用 Kali 时，优化界面显示

### Kali Tweaks
- **能干什么**: Kali 系统配置工具，调整各种系统设置
- **核心参数**: GUI 界面操作
- **实战直觉**: 快速配置 Kali 系统（网络、服务、外观等）

### Kali HiDPI Mode
- **能干什么**: 高分辨率显示器优化模式
- **核心参数**: 直接运行调整 DPI 设置
- **实战直觉**: 在 4K 或高分辨率显示器上使用 Kali 时，优化界面显示

## voiphopper
- **能干什么**: VLAN 跳跃攻击工具，针对 VoIP 网络
- **核心参数**:
  - `-i`: 网络接口
  - `-c`: CDP/LLDP 模式
- **实战直觉**: 跳到 VoIP VLAN，访问语音网络

## spooftooph
- **能干什么**: 蓝牙设备欺骗工具
- **核心参数**:
  - `-i`: 蓝牙接口
  - `-n`: 伪造设备名
- **实战直觉**: 伪造蓝牙设备，进行蓝牙攻击

## tcpreplay
- **能干什么**: 重放 pcap 数据包
- **核心参数**:
  - `-i`: 网络接口
  - pcap 文件
- **实战直觉**: 重放捕获的流量，测试 IDS/IPS 规则

## PowerShell
- **能干什么**: Windows PowerShell 命令行环境（Linux 版）
- **核心参数**:
  - `-Command`: 执行命令
  - `-File`: 执行脚本
- **实战直觉**: 在 Kali 上测试 PowerShell 脚本，准备 Windows 渗透工具

## NetHunter
- **能干什么**: Kali NetHunter 移动渗透测试平台
- **核心参数**: Android 设备上的 Kali Linux
- **实战直觉**: 移动设备渗透测试，WiFi 攻击，HID 攻击
