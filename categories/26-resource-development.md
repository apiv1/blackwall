# 资源开发 (Resource Development)

## 工具列表

### [msfvenom](../tools/msfvenom.md)
Metasploit 的 payload 生成器，创建各种格式的恶意载荷

### [searchsploit](../tools/searchsploit.md)
离线搜索 Exploit-DB 数据库

### [radare2](../tools/radare2.md)
强大的逆向工程框架，支持反汇编、调试、分析

### [pyinstaller](../tools/pyinstaller.md)
将 Python 脚本打包成独立可执行文件

### [clang/clang++](../tools/clang-clang.md)
C/C++ 编译器，用于编译 exploit 代码

### [msf-nasm_shell](../tools/msf-nasm-shell.md)
Metasploit 的汇编/反汇编交互式 shell

### [msfpc](../tools/msfpc.md)
MSF Payload Creator，快速生成各种 payload

### [generic_* (shellcode 生成器)](../tools/generic-shellcode.md)
Metasploit 的通用 shellcode 生成工具

### [generic_send_tcp](../tools/generic-send-tcp.md)
生成 TCP 发送 shellcode

### [generic_listen_tcp](../tools/generic-listen-tcp.md)
生成 TCP 监听 shellcode

### [generic_send_udp](../tools/generic-send-udp.md)
生成 UDP 发送 shellcode

### [generic_chunked](../tools/generic-chunked.md)
生成分块传输 shellcode

### [clang](../tools/clang.md)
C 语言编译器

### [clang++](../tools/clang.md)
C++ 语言编译器

### [powersploit](../tools/powersploit.md)
PowerShell 后渗透框架
