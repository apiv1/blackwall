# Web 扫描 (Web Scanning)

## 工具列表

### [ffuf](../tools/ffuf.md)
高速 Web 模糊测试工具，用于目录/文件/参数爆破

### [gobuster](../tools/gobuster.md)
多功能爆破工具，支持目录、DNS、vhost 枚举

### [dirb](../tools/dirb.md)
经典的 Web 内容扫描器，基于字典的目录爆破

### [dirbuster](../tools/dirbuster.md)
dirb 的 GUI 版本，支持多线程目录爆破

### [wfuzz](../tools/wfuzz.md)
强大的 Web 应用模糊测试框架，支持复杂的注入场景

### [lbd](../tools/lbd.md)
负载均衡检测工具，识别目标是否使用负载均衡器
