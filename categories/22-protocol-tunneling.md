# 协议隧道 (Protocol Tunneling)

## 工具列表

### [proxychains4](../tools/proxychains4.md)
强制任何程序通过代理链（SOCKS/HTTP）运行

### [dns2tcpc/dns2tcpd](../tools/dns2tcpc-dns2tcpd.md)
通过 DNS 隧道传输 TCP 连接

### [iodine (iodine-client-start)](../tools/iodine-iodine-client-start.md)
通过 DNS 创建 IPv4 隧道

### [stunnel4](../tools/stunnel4.md)
为任意 TCP 连接添加 SSL/TLS 加密

### [ptunnel](../tools/ptunnel.md)
通过 ICMP 隧道传输 TCP 连接

### [sslh](../tools/sslh.md)
多协议端口复用工具，在同一端口上运行多个服务

### [miredo](../tools/miredo.md)
Teredo IPv6 隧道客户端，通过 IPv4 NAT 建立 IPv6 连接

### [udptunnel](../tools/udptunnel.md)
通过 UDP 隧道传输 TCP 连接

### [proxytunnel](../tools/proxytunnel.md)
通过 HTTP 代理建立 SSL 隧道

### [pwnat](../tools/pwnat.md)
NAT 穿透工具，无需端口转发建立连接

### [dns2tcpc](../tools/dns2tcpc.md)
DNS 隧道客户端，通过 DNS 协议传输数据

### [dns2tcpd](../tools/dns2tcpd.md)
DNS 隧道服务端

### [iodine-client-start](../tools/iodine-client-start.md)
Iodine DNS 隧道客户端启动脚本

### [cadaver](../tools/cadaver.md)
WebDAV 客户端，类似 FTP 的文件传输

### [dbd](../tools/dbd.md)
加密的 netcat，支持 AES 加密通信
