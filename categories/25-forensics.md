# 取证工具 (Forensics)

## 工具列表

### [autopsy](../tools/autopsy.md)
图形化数字取证平台，分析磁盘镜像和文件系统

### [binwalk / binwalk3](../tools/binwalk-binwalk3.md)
固件分析工具，提取嵌入式文件系统

### [bulk_extractor](../tools/bulk-extractor.md)
从磁盘镜像中批量提取特征（邮箱、URL、信用卡号等）

### [hashdeep](../tools/hashdeep.md)
计算和验证文件哈希值，用于完整性检查

### [photorec](../tools/photorec.md)
文件恢复工具，从磁盘恢复删除的文件

### [testdisk](../tools/testdisk.md)
分区恢复和修复工具

### [scalpel](../tools/scalpel.md)
基于文件头/尾的数据雕刻工具

### [magicrescue](../tools/magicrescue.md)
使用"魔术字节"恢复文件

### [scrounge-ntfs](../tools/scrounge-ntfs.md)
NTFS 文件系统数据恢复工具

### [binwalk](../tools/binwalk.md)
固件分析工具，提取嵌入式文件系统

### [binwalk3](../tools/binwalk3.md)
binwalk 的 Python 3 版本
