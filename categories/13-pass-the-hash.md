# Pass-the-Hash 攻击 (Pass-the-Hash)

## 工具列表

### [mimikatz](../tools/mimikatz.md)
Windows 凭据提取工具，能从内存中提取明文密码和哈希

### [impacket-psexec](../tools/impacket-psexec.md)
使用哈希值远程执行命令（无需明文密码）

### [impacket-smbexec](../tools/impacket-smbexec.md)
类似 psexec，但使用不同的执行方式

### [evil-winrm](../tools/evil-winrm.md)
WinRM 协议的渗透测试工具，支持哈希认证

### [passing-the-hash](../tools/passing-the-hash.md)
Pass-the-Hash 工具集

### [netexec](../tools/netexec.md)
CrackMapExec 的继任者，用于大规模 Windows 网络渗透测试

### [smbmap](../tools/smbmap.md)
SMB 共享枚举工具，检查共享权限

### [impacket-scripts](../tools/impacket-scripts.md)
Impacket 工具集，包含多种 Windows 协议攻击工具
