# 初始访问 (Initial Access)

## 工具列表

### [sqlmap](../tools/sqlmap.md)
自动化 SQL 注入检测和利用工具，支持多种数据库

### [metasploit-framework](../tools/metasploit-framework.md)
最强大的渗透测试框架，集成漏洞利用、后渗透、payload 生成

### [commix](../tools/commix.md)
自动化命令注入和利用工具

### [setoolkit](../tools/setoolkit.md)
社会工程学工具包，用于钓鱼攻击和凭据收集

### [gophish](../tools/gophish.md)
开源钓鱼框架，用于模拟钓鱼攻击

### [dns-rebind](../tools/dns-rebind.md)
DNS 重绑定攻击工具
